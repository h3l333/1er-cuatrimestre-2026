Set of all attributes that can be inferred from an initial set using functional dependencies ([[Dependencias Funcionales]]).

>Often expressed using "X^+".

Intuitively speaking, it is "all that you can obtain from X" applying inference rules.

---
### Ejemplo

Supongamos:

- $F = \{ A \to B,\; B \to C \}$
